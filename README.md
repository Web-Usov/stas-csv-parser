# Coffee Report CLI

CLI-инструмент для формирования отчетов по потреблению кофе студентами на основе CSV-файлов.

##  Пример запуска
```
python3 main.py -f csv_examples/*.csv -r median-coffee
```
или
```
python main.py -f csv_examples/math.csv csv_examples/programming.csv csv_examples/physics.csv -r median-coffee 
```

## Пример вывода

```
| Студент           |   Траты на кофе |
|-------------------|-----------------|
| Иван Кузнецов     |             700 |
| Дмитрий Морозов   |             610 |
| Михаил Павлов     |             590 |
```

## Архитектура

* Использован паттерн Registry для выбора отчёта
* Использован паттерн Strategy для разделения логики отчета
* Каждый отчёт реализует общий интерфейс `Report[T]`
* Логика разделена на слои:

  * `reports/` — расчёты
  * `services/` — работа с данными
  * `main.py` — CLI и orchestration


## Примечание

Чтобы добавить новый отчёта требуется только создать новый класс и добавить его импорт в main.py.
